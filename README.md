# SKCT 연습 프로그램

SK그룹 인적성 검사(SKCT) 준비를 위한 답안 입력 + 보조 도구 통합 연습 앱입니다.

> **Vibe-coded with [Claude](https://claude.ai) (Anthropic)**

---

## 실행 방법

Python 3.x 기본 내장 라이브러리만 사용하므로 별도 설치 없이 실행 가능합니다.

```bash
python3 skct_practice.py
```

---

## 주요 기능

### 모드 선택

| 모드 | 설명 |
|------|------|
| **20문제 모드** | 20문제, 총 15분 제한. 문제별 소요 시간 기록 후 결과표 출력 |
| **1문제 연습 모드** | 문제당 45초 제한. 답 선택 즉시 타이머 정지, 다음 문제로 계속 진행 |

### 답안 입력

- 화면 버튼(1~5) 또는 키보드 숫자키로 답 선택
- 문제당 타이머 표시 (`00:00.0` 형식, 100ms 단위)
- 45초 초과 시 타이머 빨간색 전환
- 20문제 모드: 우측 답안 현황 인라인 표 실시간 업데이트
- 언제든 **답안 현황 보기** 버튼으로 지금까지의 답안 확인 가능

### 내장 보조 도구 (한 화면에 항상 표시)

| 도구 | 특징 |
|------|------|
| **메모장** | 자유 텍스트 입력, 한국어 IME 지원, macOS 기본 단축키 지원 |
| **그림판** | 마우스 드래그로 자유 드로잉, 지우기 버튼 |
| **계산기** | 버튼 클릭 또는 키보드 입력, 숫자 세 자리 쉼표 표시 |

### 계산기 키보드 지원

계산기 활성 상태에서 키보드로 바로 입력:

- 숫자 `0`~`9`, 넘패드 숫자
- `+` `-` `*` `/`
- `Enter` / 넘패드 `Enter` → `=`
- `Backspace` → 한 글자 삭제
- `C` → 초기화

### 기타

- 20문제 모드 15분 종료 시 알람음 3회
- 메인메뉴 복귀 시 메모장·그림판·계산기 초기화
- 활성 패널 헤더 색상으로 포커스 위치 표시
- macOS 다크모드에서도 색상 정상 렌더링

## 환경

- Python 3.x (표준 라이브러리 `tkinter`, `time`, `subprocess`)
